# 📱 Projeto Mobile - Loja de Roupas

## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo mobile simples de e-commerce utilizando React Native. O aplicativo simula uma loja de roupas onde o usuário pode realizar login, visualizar categorias de produtos e acessar detalhes de itens disponíveis.

O objetivo do projeto é aplicar os conceitos estudados em aula relacionados ao desenvolvimento de aplicativos móveis, organização de código, navegação entre telas e gerenciamento de estado.

---

## 🚀 Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas no desenvolvimento do projeto:

- React Native
- Expo
- Axios
- Redux Toolkit
- JavaScript
- Node.js

---

## 📂 Estrutura do Projeto

O projeto foi organizado da seguinte forma:

scr/
 ├── screens
 │      Login.js
 │      Masculino.js
 │      Feminino.js
 │      DetalheProduto.js
 │
 ├── services
 │      api.js
 │
 └── store
        store.js

Descrição das pastas:

screens → Contém todas as telas do aplicativo  
services → Responsável pela comunicação com APIs utilizando Axios  
store → Gerenciamento de estado utilizando Redux Toolkit

## 📱 Funcionalidades do Aplicativo

O aplicativo possui as seguintes funcionalidades:

- Tela de login com validação de usuário e senha
- Tela inicial com navegação entre categorias
- Listagem de roupas masculinas
- Listagem de roupas femininas
- Tela de detalhes do produto
- Navegação entre telas
- Logout do sistema

## ▶️ Como Executar o Projeto

1. Clonar o repositório

git clone https://github.com/JONATHA-RICHARD/react-native-loja-roupas

2. Entrar na pasta do projeto

cd react-native-loja-roupas

3. Instalar dependências

npm install

4. Executar o projeto

npx expo start

Após executar o comando, o projeto poderá ser aberto no navegador ou em um emulador.
