# Projeto Mobile - Loja de Roupas

## Descrição

Este projeto consiste no desenvolvimento de um aplicativo mobile de e-commerce utilizando React Native. O aplicativo apresenta funcionalidades básicas de navegação entre categorias de produtos, simulando uma loja de roupas com categorias masculinas e femininas.

O objetivo do projeto é aplicar os conceitos aprendidos em aula, como consumo de APIs, gerenciamento de estado com Redux Toolkit e organização de código em aplicações mobile.

---

## Tecnologias utilizadas

- React Native
- Expo
- Axios
- Redux Toolkit
- JavaScript

---

## Funcionalidades

- Tela de login
- Listagem de produtos masculinos
- Listagem de produtos femininos
- Tela de detalhes do produto
- Navegação entre telas
- Logout do sistema
